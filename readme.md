# Flask Web App Tutorial

## Setup & Installation

Pastikan kamu menginstall python terbaru, versi python yg digunakan saat membuat ini adalah 3.13.3

```bash
git clone <repo-url>
```

```bash
pip install -r requirements.txt
```

## Running The App

```bash
python main.py
```

## Viewing The App

Go to `http://127.0.0.1:1903`
